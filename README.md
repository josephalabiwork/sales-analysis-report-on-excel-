# sales-analysis-report-on-excel-
sales analysis using excel.. created pivot tables for analysis and an interactive dashboard for better data driven decisions.
Introduction

This project provides a data-driven analysis of sales performance using Microsoft Excel.
It explores transaction data across multiple Nigerian regions to uncover key insights about business performance.

Project Focus
Revenue performance across regions
Top-selling products and categories
Salesperson performance and contributions
Customer demographics and purchasing patterns
Analysis Process
Data cleaning to ensure accuracy and consistency
Creation of calculated columns for performance metrics
Use of pivot tables to summarize sales data
Design of an interactive dashboard for visual insights
Outcome
Provided clear visibility into regional and product-level performance
Highlighted top-performing salespersons and customer segments
Demonstrated how Excel supports effective business intelligence and data-driven decision making
